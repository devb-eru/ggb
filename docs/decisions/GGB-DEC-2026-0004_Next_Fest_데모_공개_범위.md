## 질문

Next Fest 참가자에게 2장까지만 공개할 것인가, 프롤로그~4장·엔딩 전체를 공개할 것인가.

## 결정

**2장까지만 공개한다.** 행사 시점에 기본 게임은 출시 후보 수준으로 완성하되, 데모 AppID의 공개 콘텐츠는 2장 종료에서 차단한다.

## 선택 이유

- 전체 게임을 무료로 공개하지 않고 구매 전환과 후반부 발견 경험을 보존한다.
- 이미 정한 2~3시간 30분 데모 플레이 시간과 QA 범위를 유지할 수 있다.
- Next Fest 피드백을 출시 후보 빌드에 반영할 안정화 시간을 확보한다.

## 영향과 후속 작업

- 데모와 기본 게임은 콘텐츠 플래그·depot·빌드 채널을 분리한다.
- 데모 종료 화면에서 기본 게임 상점과 찜하기로 연결한다.
- 행사 시점의 데모 자산과 시스템 품질은 출시 후보 기준을 사용한다.
