## 결정

대화 기록은 확인한 대사와 선택지를 순서대로 저장한다. 저장·불러오기 뒤에도 유지하며 새 게임 또는 호환 불가 하드 리셋에서만 초기화한다.

## 이유

- 사용자가 놓친 정보를 음성 없이 다시 확인할 수 있다.
- 저장·불러오기 때문에 기록이 사라지는 혼란을 막는다.
- 선택지와 서사 단서를 다시 확인하는 접근성 기능으로 작동한다.

## 구현 조건

- 저장 항목에 대사 ID, 표시 언어, 화자, 선택지, 표시 순서를 기록한다.
- 현지화 문구 자체보다 대사 ID를 저장해 언어 변경 뒤 해당 언어로 다시 표시한다.
- 아직 확인하지 않은 대사와 후속 선택지는 기록에 노출하지 않는다.
