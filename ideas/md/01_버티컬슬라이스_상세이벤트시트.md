# GGB 버티컬 슬라이스 상세 이벤트 시트

## 1. 이벤트 시트 규칙

### 이벤트 ID

- `P-XX`: 프롤로그 이벤트
- `L1-XX`: 1장 이벤트
- `L2-XX`: 2장 이벤트
- `SYS-XX`: 반복적으로 발생하는 시스템 이벤트

### 공통 필드

| 필드 | 의미 |
| --- | --- |
| 선행 조건 | 이벤트 실행에 필요한 플래그와 시간대 |
| 시작 동기 | 플레이어가 현재 무엇을 하려 한다고 이해하는지 |
| 필수 행동 | 이벤트 완료에 필요한 상호작용 |
| 정보 공개 | 새로 확정되는 사실 |
| 감정 변화 | 이벤트 전후 주인공의 감정 |
| 월드 변경 | 오브젝트, NPC, 시간대, 수첩, 균열 단계 변화 |
| 다음 목표 | 플레이어가 다음 행동을 떠올리게 하는 단서 |

## 2. 프롤로그: 완벽한 하루

### P-01. 같은 방식으로 열리는 아침

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침 / 주인공 침실 |
| 선행 조건 | 새 게임 시작 |
| 시작 동기 | 에드가의 안내를 따라 오늘의 일과를 시작한다. |
| 등장인물 | 주인공, 에드가 |
| 핵심 기능 | 이동, 조사, 대화의 기본 조작을 소개한다. |

#### 진행

1. 암전 상태에서 시계 초침과 새소리가 들린다.
2. 에드가가 커튼을 열며 주인공을 깨운다.
3. 플레이어는 침대, 창문, 아버지 사진, 낙서 수첩을 살펴볼 수 있다.
4. 에드가에게 말을 걸면 오늘은 사용인들의 일을 도와달라는 부탁을 받는다.
5. 침실 문을 사용하면 대응접실로 이동한다.

#### 감각 및 심리 연출

- 침실은 따뜻하고 안전하게 보인다.
- 아버지 사진을 조사하면 주인공은 아버지가 언제 돌아오는지 묻지 않는다. 이미 오래 기다리는 일에 익숙하다는 느낌만 남긴다.
- 수첩에는 고딕 저택과 사용인들의 과장된 그림이 있다.

#### 오브젝트 반응 예시

- 침대: `조금 전까지 자고 있었는데도, 이불은 이미 가지런하다.`
- 창문: `오늘도 좋은 날씨다. 언제나 그렇듯이.`
- 아버지 사진: `아버지는 사진을 찍을 때도 웃는 법을 몰랐다.`
- 수첩: `그릴 게 없을 때는 집을 그린다. 집은 도망가지 않으니까.`

#### 완료 및 변경

- `tutorial_look = true`
- `tutorial_talk = true`
- 대응접실과 서재 이동 해금.
- 수첩 메뉴 해금.
- 다음 목표: 마라와 에드가가 제시한 아침 일과 확인.

### P-02. 창문 닦기

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침 / 대응접실 |
| 선행 조건 | `P-01` 완료 |
| 시작 동기 | 손이 부족하다는 마라의 부탁을 돕는다. |
| 등장인물 | 주인공, 마라 |
| 핵심 기능 | 아이템 획득과 오브젝트에 아이템 사용을 익힌다. |

#### 진행

1. 마라는 창가에 청소용 천과 세정제를 놓고 주인공에게 창문 세 장을 닦아달라고 부탁한다.
2. 플레이어는 천과 세정제를 인벤토리에 넣는다.
3. 첫 번째 창문은 천만 사용하면 닦인다.
4. 두 번째 창문은 세정제를 먼저 사용한 뒤 천을 사용해야 한다.
5. 세 번째 창문을 닦는 동안 바깥의 새가 같은 궤도로 세 번 지나간다.
6. 마라에게 완료를 보고한다.

#### 감각 및 심리 연출

- 마지막 창문은 다른 창문보다 유난히 차갑다.
- 주인공은 새의 반복을 눈치채지만 자신이 오래 바라봐서 그렇게 느꼈다고 생각한다.
- 마라는 새에 관한 질문에 창문 상태만 확인하며 답을 피한다.

#### 실패와 반복 반응

- 천만 반복 사용: `얼룩이 번지기만 한다. 마라는 이런 걸 보면 한숨부터 쉴 것이다.`
- 세정제를 너무 많이 사용: 마라가 다가와 양을 조절해주며 조합 순서를 다시 암시한다.

#### 완료 및 변경

- `tutorial_item_use = true`
- 대응접실 창문 상태 `clean`.
- 수첩에 선택 기록: `창밖의 새는 같은 곳을 세 번 날았다.`
- 다음 목표: 서재의 책 정리.

### P-03. 잘못 꽂힌 책

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침 / 서재 |
| 선행 조건 | `P-01` 완료 |
| 시작 동기 | 에드가의 부탁으로 책 세 권을 제자리에 꽂는다. |
| 등장인물 | 주인공, 에드가 |
| 핵심 기능 | 조사 정보를 이용한 간단한 정렬 퍼즐을 소개한다. |

#### 진행

1. 에드가는 책 세 권의 책등 문양을 보여주고 각 책장 구역의 장식과 맞춰 꽂으라고 한다.
2. 플레이어는 책등의 장미, 새, 시계 문양을 책장의 같은 문양과 대응시킨다.
3. 시계 문양 책을 꽂으면 뒤쪽에서 낡은 일지가 떨어진다.
4. 일지는 대부분 번져 읽을 수 없고, 표지 안쪽에 어린 주인공의 낙서가 남아 있다.
5. 에드가는 일지를 발견하면 잠시 굳지만 곧 오래된 장부라고 설명한다.

#### 감각 및 심리 연출

- 일지 표면은 먼지가 쌓였지만 손에 묻어나지 않는다.
- 주인공은 일지의 종이가 이상하게 따뜻하다고 느낀다.
- 에드가는 일지를 주인공에게서 빼앗지 않는다. 대신 제자리에 두도록 지나치게 부드럽게 권한다.

#### 완료 및 변경

- `tutorial_observation_puzzle = true`
- 아버지 일지 오브젝트 해금. 현재 상태는 `unreadable`.
- 수첩에 `아버지의 일지? 에드가는 장부라고 했다.` 기록.
- 다음 목표: 주방에서 차 준비.

### P-04. 아버지가 좋아하던 차

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 낮 / 주방 |
| 선행 조건 | `P-02`, `P-03` 완료 |
| 시작 동기 | 루카의 지시에 따라 오후 차를 준비한다. |
| 등장인물 | 주인공, 루카, 짧게 등장하는 에드가 |
| 핵심 기능 | 아이템 조합과 선택 대화를 소개한다. |

#### 진행

1. 루카는 찻잎, 뜨거운 물, 우유를 준비해달라고 한다.
2. 플레이어는 찻잎과 물을 먼저 조합한 뒤 우유를 더한다.
3. 설탕을 넣을지 선택할 수 있다.
4. 루카는 어떤 선택을 해도 `아버지는 설탕을 넣지 않았다`고 말한다.
5. 주인공은 아버지에 관한 질문 하나를 선택한다.

#### 선택 대화

- `아버지는 언제 돌아와?`
- `아버지는 여기서 무슨 일을 했어?`
- `왜 다들 아버지 취향을 나보다 잘 알아?`

선택은 사건 진행을 바꾸지 않지만, 루카의 대답과 주인공 수첩 문장이 달라진다.

#### 감각 및 심리 연출

- 차의 향은 분명하지만 맛을 묘사하는 독백은 없다.
- 루카가 찻잔을 건네는 순간 주인공에게 짧은 졸음이 몰려온다.
- 루카는 이를 보고 컵을 바로 회수하며 `오늘은 일찍 쉬는 편이 좋겠다`고 말한다.

#### 완료 및 변경

- `tutorial_combine = true`
- `father_dialogue_choice` 저장.
- 시간대가 저녁으로 전환.
- 다음 목표: 저녁 종이 울리기 전 자유 조사.

### P-06. 모두가 재촉하는 밤

| 시간 / 장소 | 밤 / 저택 공용 공간과 침실 |

| 핵심 기능 | 첫 리셋 직전의 불편함을 조성한다. |

#### 진행

1. 저녁 종이 울리는 순간 사용인들의 위치가 일제히 바뀐다.
2. 누구에게 말을 걸어도 표현만 다를 뿐 침실로 돌아가라는 답을 한다.
3. 주인공이 다른 방으로 이동하려 하면 에드가가 복도 끝에 나타나 부드럽게 길을 막는다.
4. 침실에서 침대를 사용하면 취침 확인 대사가 나온다.

- 자장가 아래에 일정한 심박음이 섞인다.
- 주인공은 잠들고 싶지 않다고 생각하지만 몸에서 힘이 빠진다.

#### 완료 및 변경

- `loop_index = 1`
- 일반 오브젝트 상태 초기화.
- 수첩과 플레이어 지식 유지.
- `SYS-01 첫 리셋` 실행.

## 3. 1장: 어제의 흔적

### SYS-01. 첫 리셋

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침 / 침실 |
| 선행 조건 | `P-06` 완료 |
| 시작 동기 | 같은 아침이 반복되었는지 확인한다. |
| 등장인물 | 주인공, 에드가 |

#### 진행

1. 프롤로그와 동일한 카메라, 새소리, 에드가의 인사로 시작한다.
2. 에드가의 첫 대사는 프롤로그와 완전히 동일한 음성 또는 타이밍으로 재생한다.
3. 주인공은 대사 도중 먼저 다음 말을 떠올린다.
4. 플레이어가 방을 조사하면 전날과 같은 상태로 돌아간 것을 확인한다.
5. 수첩만은 전날 기록을 유지한다.

#### 감정 변화

- 단순한 기시감에서 `무언가 잘못되었다`는 인식으로 이동한다.
- 에드가에게 바로 묻거나 혼자 확인하는 선택을 제공한다.

#### 에드가에게 질문할 경우

- 에드가는 꿈을 꾼 것이라 설명한다.
- 주인공이 창문과 책 이야기를 구체적으로 말하면 에드가는 아주 짧게 대답을 늦춘다.

#### 완료 및 변경

- `loop_awareness = suspected`
- 새 목표: 어제가 실제로 존재했는지 증명한다.

### L1-01. 남는 표시

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침~밤 / 침실 |
| 선행 조건 | `SYS-01` 실행 |
| 시작 동기 | 주인공이 자신의 기억이 착각인지 검증한다. |
| 핵심 기능 | 수첩은 리셋되지 않는다는 규칙을 확정한다. |

#### 진행

1. 수첩을 조사하면 주인공이 새로운 표시를 남길 수 있다.
2. 표시 문구는 `내일 아침, 이 문장을 읽어.`로 고정한다.
3. 이번 루프의 일과는 축약 수행할 수 있다.
4. 밤에 취침하고 다음 아침 수첩을 확인한다.
5. 표시가 남아 있으며 주인공이 루프를 확신한다.

#### 유연한 시간제 처리

- 이미 완료한 프롤로그 잡일은 `직접 수행`과 `빠르게 돕는다` 중 선택 가능하다.
- 빠르게 돕기를 선택하면 시간대가 전환되고 핵심 보상은 동일하게 제공된다.

#### 완료 및 변경

- `loop_index = 2`
- `loop_awareness = confirmed`
- `notebook_persists = true`
- 수첩에 `나는 어제를 기억한다. 수첩도 기억한다.` 기록.
- 다음 목표: 반복을 이용해 에드가가 지키는 서재를 조사한다.

### L1-02. 사용인 시간표

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침~저녁 / 저택 전역 |
| 선행 조건 | `loop_awareness = confirmed` |
| 시작 동기 | 전날에는 들어갈 수 없었던 서재 제한 구역에 접근한다. |
| 핵심 기능 | 루프를 정보 수집 도구로 사용한다. |

#### 진행

1. 아침의 서재에는 에드가가 있어 일지에 오래 접근할 수 없다.
2. 플레이어는 사용인들에게 말을 걸거나 수첩에 위치를 기록할 수 있다.
3. 낮에는 에드가가 주방에서 루카와 대화한다.
4. 저녁 종 직전에는 서재가 비어 있다.
5. 저녁의 서재에서 아버지 일지를 조사하면 첫 문장이 복원된다.

#### 보조 단서

- 마라는 에드가가 저녁마다 `시계를 확인하러 간다`고 말한다.
- 루카는 오후 차를 준비하면 에드가가 반드시 주방에 온다고 알려준다.
- 이리스는 온실 유리 너머로 에드가가 서쪽 복도를 지나가는 시간이 일정하다고 말한다.

#### 완료 및 변경

- `servant_schedule_known = true`
- `journal_stage = 1`

### L1-03. 열세 번째 종

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 하루 전체 / 침실, 대응접실, 서재, 서쪽 복도 |
| 선행 조건 | `journal_stage = 1` |
| 시작 동기 | 아버지의 메시지에 따라 틀린 시계를 찾는다. |
| 핵심 기능 | 여러 시간대의 관찰 정보를 한 퍼즐로 결합한다. |

#### 퍼즐 구성

| 시계 | 표면 상태 | 관찰 정보 |
| --- | --- | --- |
| 침실 탁상시계 | 한 시간 느림 | 종이 울리지 않는다. |
| 대응접실 괘종시계 | 정상처럼 보임 | 저녁에 열두 번 울린다. |
| 서재 벽시계 | 멈춰 있음 | 가까이 가면 아주 약한 진동이 느껴진다. |
| 서쪽 복도 대시계 | 장식물처럼 보임 | 저녁 종 이후 아무도 없는 순간 열세 번째 소리를 낸다. |

#### 진행

1. 수첩의 `시간 기록` 페이지가 해금된다.
2. 플레이어는 각 시간대에 시계를 조사해 차이를 기록한다.
3. 조사한 시계 중 하나의 봉인핀을 해제해 저녁 시계망을 작동시킨다.
4. 한 시계가 작동하면 다른 시계는 안전 잠금에 들어가 그날 다시 시험할 수 없다.
5. 잘못된 시계를 선택하면 열두 번의 종 또는 불완전한 금속음만 들린다.
6. 주인공은 선택한 시계와 결과를 수첩에 기록하고 잠들어 물리 상태를 초기화한다.
7. 다음 아침에는 같은 침실에서 시작하지만 해결한 일과, 시간표 조사, 일지 1단계는 숏컷으로 통과할 수 있다.
8. 올바른 시계를 선택하면 사용인들의 대화가 멈춘 정적 뒤 열세 번째 금속음이 울린다.
9. 수첩에 열세 번째 종의 파형이 기록되고 아버지 일지의 표식이 같은 진동으로 반응한다.
10. 서재에서 일지를 다시 접촉하면 두 번째 문장이 영구 복원된다.

#### 감각 및 심리 연출

- 열세 번째 종은 귀로만 들리지 않고 주인공의 갈비뼈 안쪽에서 울리는 것처럼 묘사한다.
- 종이 울린 순간 복도 조명이 한 박자 늦게 깜박인다.
- 잘못된 시계가 잠길 때 내부 톱니가 한 방향으로만 무너지는 소리를 들려주어 당일 복구가 불가능함을 전달한다.

#### 실패와 리셋

- 실패는 반사 신경이나 무작위 결과가 아니라 플레이어가 선택한 시계에 따라 결정된다.
- 실패한 시계 ID, 종의 횟수, 잠금 반응은 수첩에 유지된다.
- 봉인핀, 시계 잠금, 시간대는 잠들면 초기화된다.
- 다음 루프에는 이미 확인한 시계를 제외하거나 수첩의 연결 스케치로 새 시계를 선택할 수 있다.
- 실패 후에도 아침부터 시작하며 B3 앞으로 직접 이동하는 체크포인트는 사용하지 않는다.

#### 완료 및 변경

- `thirteenth_bell_known = true`
- `journal_stage = 2`
- `tested_clock_ids`와 `clock_network_rule_known` 유지.
- 일지 문장: `열세 번째 종이 울리면 검은 거울의 얼룩을 지워라.`
- 대응접실 검은 거울에 새로운 조사 반응 추가.
- 일지 2단계 복원 후 잠들면 `loop_index`가 증가하고 C단계 아침이 시작된다.

### L1-04. 금지의 확인

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 아침 / 대응접실 |
| 선행 조건 | `journal_stage = 2` |

#### 진행

1. 검은 거울을 조사하면 표면의 얼룩이 일반 먼지와 다르다는 반응이 추가된다.
2. 에드가는 주인공이 거울에 가까이 가는 순간 나타난다.
3. 에드가는 처음으로 명확한 금지 명령을 내린다.
4. 주인공이 이유를 물으면 에드가는 거울이 오래되어 위험하다고 답한다.

6. 마라는 주인공에게 거울을 닦을 생각이라면 일반 세정제로는 안 된다고 무심코 말한 뒤 말을 멈춘다.

#### 감정 변화

- 아버지의 메시지를 믿고 싶은 마음과 에드가를 배신하는 듯한 죄책감이 충돌한다.
- 에드가가 거울을 알고 두려워한다는 사실 때문에 확인 욕구가 더 강해진다.

#### 완료 및 변경

- `mirror_forbidden = true`
- 마라에게 `거울의 얼룩` 대화 주제 해금.
- 2장 시작.

## 4. 2장 초반: 금지된 거울

| 항목 | 내용 |

- 마라가 도와주는 것인지 막는 것인지 불분명하게 연출한다.

#### 완료 및 변경

### L2-02. 열세 번째 종의 검은 거울

| 항목 | 내용 |
| --- | --- |
| 시간 / 장소 | 저녁 / 대응접실 |
| 선행 조건 | `neutral_cleaner_ready = true`, `thirteenth_bell_known = true` |
| 시작 동기 | 아버지의 메시지에 따라 거울의 얼룩을 지운다. |
| 등장인물 | 주인공, 마지막에 에드가 |
| 핵심 기능 | 슬라이스의 고점. 고딕 저택과 SF 시설의 첫 직접 충돌. |

#### 진행

1. 저녁 종 전 세정제를 사용하면 검은 막이 용액을 흡수해 단단히 굳으며, 그날에는 다시 닦을 수 없다.
2. 열두 번의 종이 끝난 뒤 대응접실의 모든 생활 소리가 멈춘다.
3. 열세 번째 종이 울리면 거울 표면에 얇은 선이 나타난다.
4. 플레이어가 중성 세정제와 부드러운 천을 사용한다.
5. 닦는 동작에 맞춰 검은 막이 유리 얼룩이 아니라 렌더링 레이어처럼 사라진다.
6. 거울 뒤에서 청백색 진단 패널과 회로 문양이 드러난다.
7. 패널에 읽을 수 없는 시스템 문구와 지하 좌표가 짧게 나타난다.
8. 거울 속 주인공의 반사가 한 박자 늦게 움직인다.
9. 반사가 주인공의 방이 아닌 냉각 장치 내부에 누운 소녀의 실루엣으로 바뀐다.
10. 주인공이 거울에 손을 대면 유리가 아니라 차가운 금속의 촉감이 느껴진다.
11. 뒤에서 에드가가 처음으로 역할극을 잃은 목소리로 주인공의 이름 대신 `대상`이라는 단어를 말한다.
12. 화면이 짧게 글리치된 뒤 에드가가 평소 말투로 돌아와 거울에서 떨어지라고 명령한다.

#### 실패와 리셋

| 실패 | 당일의 돌이킬 수 없는 결과 | 다음 루프에 유지되는 정보 |
| --- | --- | --- |
| 열세 번째 종 전에 용액 사용 | 코팅이 굳어 더 이상 닦이지 않음 | 종이 울릴 때 코팅이 느슨해진다는 추론 |
| 천 없이 용액을 직접 도포 | 용액이 코팅 위에서 막을 형성하고 모두 소모됨 | 천에 소량 묻혀 선을 따라야 한다는 기록 |
| 에드가가 있는 시간에 시도 | 에드가가 천과 용액을 회수함 | 에드가가 거울을 비우는 정확한 시간 |

- 실패 후 주인공은 잠을 선택해 같은 아침으로 돌아간다.
- 코팅, 세정제 아이템, NPC 위치는 초기화된다.
- 일지 2단계, 제조법, 실패 원인, 해결한 일과의 숏컷은 유지된다.
- 다음 루프에는 재료를 빠르게 확보하고 다른 방식으로 C4를 재시도할 수 있다.

#### 주인공 감정 흐름

1. 들킬지 모른다는 긴장.
2. 아버지의 흔적을 찾았다는 짧은 안도.
3. 거울이 거울이 아니라는 감각적 공포.
4. 냉각 장치 속 자신을 본 뒤의 현실 부정.
5. 에드가의 낯선 말투로 인한 배신감.
6. 두려움에도 불구하고 지하 좌표를 기억하려는 의지.

#### 슬라이스 종료 연출

- 에드가가 거울 앞을 가리지만, 수첩에는 주인공이 보지 않은 사이 지하 좌표와 `심장은 집의 가장 낮은 곳에서 뛴다`라는 문장이 적힌다.
- 대응접실 벽 장식 일부가 한 프레임 동안 시설 배관으로 바뀐다.
- 에드가가 `오늘 본 것은 잊으셔야 합니다`라고 말한다.
- 주인공의 마지막 독백: `잊으면, 다시 같은 아침이 올 것이다. 이번에는 잊지 않는다.`
- 타이틀 또는 `버티컬 슬라이스 종료` 표시.

#### 완료 및 변경

- `mirror_revealed = true`
- `journal_stage = 3`
- `glitch_stage = 2`
- `basement_coordinates_known = true`
- 다음 본편 목표: 지하창고 입구를 찾는다.

## 5. 반복 시스템 이벤트

### SYS-02. 이미 본 일과 축약

- 한 번 완료한 튜토리얼 잡일은 다음 루프부터 축약 가능하다.
- 축약 선택 시 주인공이 자동으로 일을 처리하고 핵심 시간대만 전환한다.
- 처음과 다른 점이 생긴 잡일은 축약 버튼에 `무언가 다르다` 표시를 추가한다.

### SYS-03. 같은 대사 반복

- 동일한 대사를 두 번째 들을 경우 빠르게 넘길 수 있다.
- 세 번째 반복부터 주인공의 짧은 비교 독백을 추가한다.
- 정확히 같은 음성과 타이밍이 반복되는 대사는 의도된 불안 요소로 사용한다.

### SYS-04. 목표를 잃은 플레이어 힌트

1. 주인공의 감각 독백.
2. 수첩의 관련 항목 강조.
3. 마라 또는 루카의 간접 힌트.
4. 에드가의 금지 대사로 역방향 힌트 제공.
