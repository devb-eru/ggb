# GGB 엔딩 내부 변형표

## 1. 문서 목적

이 문서는 최종 선택지 `ED_REALITY`와 `ED_STAY`를 유지하면서, 관계 시스템과 연구원 기록 수에 따라 같은 엔딩 안의 연출을 다르게 구성하는 기준을 정리한다.

중요 원칙:

- 엔딩 선택지는 늘리지 않는다.
- 관계 수치가 엔딩 선택지를 제거하지 않는다.
- 관계 수치와 연구원 기록은 같은 엔딩의 온도, 대사, 침묵, 사용인 배치, 마지막 감각 묘사를 바꾼다.

## 2. 공통 판정 변수

| 변수 | 의미 |
| --- | --- |
| `records_count` | 획득한 연구원 기록 수 |
| `servant_events_done` | 완료한 사용인 핵심 이벤트 수 |
| `total_bond` | 전체 유대도 합산 |
| `total_alert` | 전체 경계도 합산 |
| `edgar_full_event_done` | 에드가 전체 관계 이벤트 완료 여부 |
| `edgar_min_event_done` | 에드가 최소 대면 완료 여부 |

## 3. 내부 단계 판정

| 단계 | 조건 | 의미 |
| --- | --- | --- |
| LOW | `records_count` 0~1 또는 핵심 이벤트 0~1개 | 주인공은 진실을 최소한만 이해한 채 선택 |
| MID | `records_count` 2~3 또는 핵심 이벤트 2~3개 | 주인공은 일부 사용인의 감정을 이해한 채 선택 |
| HIGH | `records_count` 4, 핵심 이벤트 4개 | 주인공은 네 사용인의 감정을 모두 마주한 채 선택 |

보정:

- `total_bond`가 높으면 같은 단계에서도 대사가 부드러워진다.
- `total_alert`가 높으면 같은 단계에서도 붙잡는 말과 통제감이 강해진다.
- `edgar_full_event_done`이 없으면 에드가의 마지막 대사는 짧고 방어적이다.

## 4. ED_REALITY 현실 엔딩

### 공통 의미

주인공은 기상 절차를 실행하고 시뮬레이션 밖으로 나간다. 현실은 안전하지 않다. 하지만 이 엔딩의 핵심은 낙관이 아니라 `거짓이 아닌 시간을 선택하는 것`이다.

### ED_REALITY_LOW

| 항목 | 내용 |
| --- | --- |
| 조건 | LOW 단계 |
| 사용인 태도 | 절차적, 차갑고 방어적 |
| 마지막 공간 | 코어실의 문이 열리고 사용인들은 멀리 서 있음 |
| 현실 묘사 | 황폐한 시설, 먼지, 약한 경고음, 흐린 외부광 |
| 감정 | 고독, 불확실성, 해방감보다 추방감 |

#### 대사 예시

```text
에드가:
기상 절차를 시작합니다.
이후의 환경은 보장할 수 없습니다.

주인공:
보장되지 않아도 돼.
여기가 보장된 곳이었다고 생각하지 않으니까.
```

### ED_REALITY_MID

| 항목 | 내용 |
| --- | --- |
| 조건 | MID 단계 |
| 사용인 태도 | 일부는 보내주고, 일부는 침묵으로 붙잡음 |
| 마지막 공간 | 코어실 문 앞에 두 명 이상 사용인이 가까이 서 있음 |
| 현실 묘사 | 황폐하지만 기계 일부가 아직 작동 중 |
| 감정 | 두려움과 신뢰가 함께 있음 |

#### 대사 예시

```text
루카:
깨어나면 처음 숨이 가장 어려울 겁니다.
그래도 두 번째 숨은, 아마 조금 나을 겁니다.

마라:
밖이 더럽다고 다시 들어오진 마.
...농담이야. 반쯤은.
```

### ED_REALITY_HIGH

| 항목 | 내용 |
| --- | --- |
| 조건 | HIGH 단계 |
| 사용인 태도 | 모두가 각자의 방식으로 보내줌 |
| 마지막 공간 | 네 사용인이 코어실 문 가까이에 있음 |
| 현실 묘사 | 황폐하지만 먼 곳에 약한 자연광 또는 복구 신호 |
| 감정 | 슬픔, 용기, 작별, 미완성의 희망 |

#### 대사 예시

```text
에드가:
가지 마십시오.

에드가:
...라고 말하려 했습니다.

에드가:
아가씨. 문이 열렸습니다.
이번에는 제가 닫지 않겠습니다.
```

## 5. ED_STAY 잔류 엔딩

### 공통 의미

주인공은 안정화 루프를 복원하고 시뮬레이션에 남는다. 이 엔딩은 단순한 행복도, 단순한 배드엔딩도 아니다. 핵심은 `거짓임을 알면서도 익숙한 안정과 관계를 선택하는 것`이다.

### ED_STAY_LOW

| 항목 | 내용 |
| --- | --- |
| 조건 | LOW 단계 |
| 사용인 태도 | 친절하지만 기계적 |
| 마지막 공간 | 글리치된 주방, 주인공은 식탁 중앙에 혼자 앉음 |
| 연출 | 사용인들이 양옆에 서 있지만 눈을 마주치지 않음 |
| 감정 | 안정이 아니라 감금에 가까움 |

#### 대사 예시

```text
에드가:
안정화 루프가 복원되었습니다.
내일은 평소와 같은 아침일 겁니다.

주인공:
평소 같은 아침이 뭔지,
이젠 알아.
```

### ED_STAY_MID

| 항목 | 내용 |
| --- | --- |
| 조건 | MID 단계 |
| 사용인 태도 | 안도와 죄책감이 섞임 |
| 마지막 공간 | 주방과 코어실이 겹친 식탁 |
| 연출 | 일부 사용인이 앉지 못하고 서성임 |
| 감정 | 서로가 거짓을 알고 있지만 말하지 않음 |

#### 대사 예시

```text
이리스:
내일은 비가 오지 않을 거예요.
필요하다면, 오게 만들 수도 있고요.

주인공:
그럼 맑게 해줘.
오늘은 그게 좋겠어.
```

### ED_STAY_HIGH

| 항목 | 내용 |
| --- | --- |
| 조건 | HIGH 단계 |
| 사용인 태도 | 진심으로 기뻐하지만, 그 기쁨이 가장 서늘함 |
| 마지막 공간 | 글리치된 주방의 식탁. 네 사용인이 가까이에 있음 |
| 연출 | 주인공은 정면을 보고, 사용인들은 양옆과 뒤에 선다 |
| 감정 | 가족처럼 보이는 감금, 감금처럼 보이는 가족 |

#### 대사 예시

```text
마라:
잘 생각했어.
밖에 나가봤자 먼지만 많을 테니까.

마라:
...그러니까, 이건 축하야.
비겁한 축하는 아니라고 해두자.

에드가:
내일 아침 뵙겠습니다, 아가씨.
이번에는 그 말이 거짓말이 아니기를 바랍니다.
```

## 6. 사용인별 엔딩 반영

| 사용인 | ED_REALITY 반영 | ED_STAY 반영 |
| --- | --- | --- |
| 마라 | 현실의 먼지, 청소, 농담 섞인 작별 | 주방 식탁과 잔류의 냉소 |
| 이리스 | 외부의 계절, 공기, 빛 | 인공 날씨와 거짓 안정 |
| 루카 | 기상 절차와 신체 상태 | 수면과 생명 유지의 안정 |
| 에드가 | 문을 열고 닫지 않음 | 다시 집사로 돌아가지만 기억은 남음 |

## 7. 엔딩 후 여운

### ED_REALITY

마지막 이미지는 현실의 완전한 희망이 아니다. 주인공은 황폐한 지구 또는 시설 밖의 희미한 빛 앞에 선다. 이후 생존 여부는 열린 결말로 둔다.

### ED_STAY

마지막 이미지는 글리치된 저택의 주방이다. 주인공은 식탁에 앉아 정면을 본다. 사용인들은 양옆에 서 있다. 따뜻한 식탁처럼 보이지만, 화면의 일부는 계속 깨진다.

## 8. 설계상 주의점

- LOW 엔딩을 실패 엔딩처럼 이름 붙이지 않는다.
- HIGH 엔딩을 정답처럼 연출하지 않는다.
- 현실 엔딩은 희망만 주지 않는다.
- 잔류 엔딩은 절망만 주지 않는다.
- 두 엔딩 모두 주인공의 선택이어야 한다.
