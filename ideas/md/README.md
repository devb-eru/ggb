## 권장 읽기 순서

1. 개요 문서에서 슬라이스의 시작·종료 상태와 핵심 질문을 확인한다.
2. 상세 이벤트 시트로 전체 플레이 흐름을 검토한다.
3. 루프 상태표를 보며 이벤트 사이의 상태 변화가 일치하는지 확인한다.
4. 퍼즐 문서에서 각 이벤트가 실제 플레이 행동으로 성립하는지 확인한다.
5. 대사 초안으로 감정선과 사용인별 말투가 의도와 맞는지 확인한다.
6. 이벤트 흐름도와 오브젝트 반응표로 장면 연결과 반복 탐색 밀도를 확인한다.
7. 검토 및 후속 작업 문서에서 수정 우선순위를 정한다.

## 현재 확정한 결정

- 전체 사용인 4명은 유지한다.
- 이번 슬라이스에서는 에드가와 마라를 중심 인물로 사용한다.
- 필수 흐름은 4개 루프 단계를 기준으로 하며, 퍼즐 실패 시 추가 리셋이 발생할 수 있다.
- 모든 리셋은 같은 침실의 같은 아침으로 돌아가며 체크포인트를 사용하지 않는다.
- 이미 본 잡일은 영구 정보와 숏컷 권한으로 축약할 수 있다.
- 메인 사건 진행은 직선형으로 유지한다.
- 금지된 거울을 닦는 동기는 아버지의 일지와 에드가의 금지가 충돌하며 형성된다.
- 슬라이스는 냉각 장치 속 주인공의 실루엣과 지하 좌표가 노출되는 순간 종료한다.
- 사용인 상호작용은 메인 진행을 막는 필수 게이트가 아니라 관계 시스템으로 운용한다.
- 연구원 기록 3개 이상은 필수 조건이 아니라 권장 진행 조건으로 유지한다.
- 에드가는 코어 접근 전 최소 대면이 필요한 반필수 인물로 둔다.
- D5 이후 잠들기는 정상 리셋이 아니라 `BROKEN_RESET`을 여는 핵심 트리거로 사용한다.
- 기획서 v0.3 초안은 md 문서로 작성되었으며, 후속 작업은 이벤트 대사·퍼즐 상세·오브젝트 반응·UI·엔딩 콘티 상세화다.
