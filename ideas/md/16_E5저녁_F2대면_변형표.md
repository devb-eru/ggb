# GGB E5 저녁 및 F2 연구원 대면 변형표

## 1. 문서 목적

이 문서는 E5 `마지막으로 정상인 저녁`과 F2 `연구원들과의 대면`을 관계 시스템의 결과에 따라 변형하는 기준을 정리한다.

두 장면의 역할은 아래와 같다.

- E5: 사용인 관계 이벤트의 중간 결산.
- F2: 연구원 기록과 유대도/경계도의 최종 결산.

## 2. 변형 단계 기준

| 단계 | 조건 | 감정 방향 |
| --- | --- | --- |
| LOW | 연구원 기록 0~1개, 핵심 이벤트 0~1개 | 차갑고 기능적. 사용인들이 다시 역할로 숨음 |
| MID | 연구원 기록 2~3개, 핵심 이벤트 2~3개 | 어색하지만 진심이 새어 나옴 |
| HIGH | 연구원 기록 4개, 핵심 이벤트 4개 | 작별 또는 잔류를 예감하는 고밀도 장면 |

보조 조건:

- `total_bond`가 높으면 같은 단계 안에서도 대사가 부드러워진다.
- `total_alert`가 높으면 같은 단계 안에서도 통제감과 불안이 강해진다.
- 에드가 전체 이벤트 미완료 시 E5와 F2에서 에드가의 발화가 짧아진다.

## 3. E5 저녁 변형

### E5_LOW: 기능적이고 차가운 저녁

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 0~1개 또는 관계 이벤트 거의 미진행 |
| 식탁 배치 | 주인공만 앉아 있고 사용인들은 벽 가까이에 선다 |
| 조명 | 고딕 저택의 따뜻한 촛불이지만 그림자가 지나치게 길다 |
| 대사 | 사용인들은 식사 절차, 수면, 코어 접근 준비만 말한다 |
| 감정 | 보호받는 느낌보다 감시받는 느낌이 강하다 |
| 기능 | 관계 이벤트를 생략한 플레이어에게 정보 공백을 체감시킴 |

#### 대사 예시

```text
에드가:
식사가 끝나면 서쪽 복도로 안내하겠습니다.

마라:
접시는 그대로 두세요. 이번엔 제가 치우죠.

루카:
식사는 조금만 드십시오. 깨어나는 절차에 방해가 될 수 있습니다.

이리스:
오늘 저녁은 맑습니다. 안쪽에서는요.
```

### E5_MID: 어색하지만 진심이 보이는 저녁

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 2~3개 또는 핵심 이벤트 2~3개 완료 |
| 식탁 배치 | 사용인 일부가 식탁 가까이에 선다 |
| 조명 | 촛불 사이에 SF 조명 노이즈가 섞인다 |
| 대사 | 한두 명이 역할 밖의 말을 하려다 멈춘다 |
| 감정 | 불편하지만 서로를 사람으로 보기 시작함 |
| 기능 | 권장 진행 조건을 만족한 플레이어에게 감정 보상 제공 |

#### 대사 예시

```text
마라:
이번엔 접시를 제대로 놨네.
...아니, 별 뜻은 없어.

루카:
깨어나는 건 아플 수 있습니다.
하지만 계속 자는 것도, 아프지 않은 건 아닙니다.

이리스:
밖의 계절은 아직 이름이 없습니다.
그래도 계절은, 이름이 없어도 지나가요.
```

### E5_HIGH: 작별의 예감이 있는 저녁

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 4개, 핵심 이벤트 4개 완료 |
| 식탁 배치 | 네 사용인이 모두 식탁 가까이에 있다 |
| 조명 | 고딕 촛불과 코어의 푸른 조명이 함께 흔들린다 |
| 대사 | 모두가 주인공에게 직접적인 감정을 남긴다 |
| 감정 | 가족처럼 보이지만, 그 친밀함이 동시에 감금처럼 느껴짐 |
| 기능 | 최종 선택 직전 가장 강한 양가감정 제공 |

#### 대사 예시

```text
에드가:
오늘의 식사는 준비되었습니다.
그리고... 내일의 식사는 준비하지 못했습니다.

마라:
밖에 나가면 먼지가 많을 거야.
그래도 네가 닦아야 할 건, 이제 여기엔 없어.

루카:
천천히 숨 쉬세요.
현실에서도 그건 아마 쓸모가 있을 겁니다.

이리스:
문밖의 계절은 친절하지 않을 거예요.
하지만 진짜 계절은 원래 허락을 구하지 않아요.
```

## 4. F2 연구원 대면 변형

### F2_LOW: 원망 중심 대면

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 0~1개 |
| 등장 방식 | 사용인 외형이 거의 벗겨지지 않고, 기계적 실루엣만 드러남 |
| 대화 태도 | 연구원들이 주인공보다 아버지를 향해 말함 |
| 정보량 | 아버지의 배신과 연구원들의 감금만 간단히 공개 |
| 감정 | 주인공은 재판장에 불려온 사람처럼 느낀다 |

#### 대사 예시

```text
연구원 음성:
그는 우리에게 미래를 약속했다.
그리고 우리를 집 안에 두었다.

에드가:
아가씨는 죄가 없습니다.
하지만 죄가 없는 사람이 문을 열 수는 있습니다.
```

### F2_MID: 갈등과 이해가 섞인 대면

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 2~3개 |
| 등장 방식 | 일부 사용인의 연구원 실루엣이 겹쳐 보임 |
| 대화 태도 | 원망과 애틋함이 번갈아 드러남 |
| 정보량 | 각 연구원의 일부 사연과 아버지의 미완성 약속 공개 |
| 감정 | 주인공은 피해자이면서 동시에 열쇠라는 위치를 이해함 |

#### 대사 예시

```text
마라:
네가 만든 감옥은 아니지.
그런데 네가 아니면 문이 안 열려.
정말 웃기지 않아?

루카:
우리는 너를 살렸고,
너를 핑계로 우리도 계속 살아 있다고 믿었습니다.
```

### F2_HIGH: 솔직한 대면

| 항목 | 내용 |
| --- | --- |
| 조건 | 연구원 기록 4개, 사용인 핵심 이벤트 모두 완료 |
| 등장 방식 | 사용인 외형, 연구원 실루엣, 기계 외피가 동시에 겹침 |
| 대화 태도 | 각 사용인이 자기 감정을 직접 말함 |
| 정보량 | 붙잡은 이유, 보내주지 못한 이유, 그래도 선택권을 주는 이유 공개 |
| 감정 | 주인공은 마지막으로 관계의 이름을 직접 정해야 함 |

#### 대사 예시

```text
에드가:
가지 마십시오.
...라고 말하도록 만들어져 있었습니다.

에드가:
가지 마십시오.
...라고 말하고 싶은 것도 사실입니다.

이리스:
문밖은 아직 사람이 살 만한 계절이 아니에요.
하지만 이 안은 너무 오래 같은 계절이었죠.

마라:
우린 네 아버지를 미워했어.
너를 미워하려고도 했고.
그게 잘 안 됐다는 게 제일 짜증 나.
```

## 5. 장면 기능 비교

| 장면 | 기능 | 플레이어에게 남길 질문 |
| --- | --- | --- |
| E5 | 사용인들과 함께 보낸 반복의 의미를 정리 | 이곳은 집인가, 감옥인가 |
| F2 | 연구원들의 원망과 애착을 직접 대면 | 그들은 가해자인가, 피해자인가 |
| ENDCHOOSE | 주인공의 선택 | 거짓된 안정과 불확실한 현실 중 무엇을 택할 것인가 |

## 6. 구현 메모

E5와 F2는 별도 엔딩을 만들지 않는다. 대신 아래 내부 상태를 참조해 연출만 바꾼다.

```yaml
ending_context:
  records_count: 3
  servant_events_done: 3
  total_bond: 7
  total_alert: 5
  edgar_full_event_done: false
  edgar_min_event_done: true
```

권장 판정:

| 판정 | 조건 |
| --- | --- |
| LOW | `records_count <= 1` 또는 `servant_events_done <= 1` |
| MID | `records_count` 2~3 |
| HIGH | `records_count == 4` 그리고 핵심 이벤트 4개 완료 |
