## 5. 루프 상태

초기화:

- 시간대.
- 인벤토리.
- 물리 색 흔적.
- 문과 장치 위치.
- 당일 NPC 위치.
- 개입 예산.

유지:

- 수첩과 자기 작성 표시.
- 복원 문장.
- 퍼즐 검증 부분.
- 관계와 연구원 기록.
- 색상 서명 식별.
- 숏컷 권한.

## 7. J4·결산 판정

```text
records 0~1 → J4_BASE
records 2~4 → J4_EXPANDED
records 5   → J4_FULL

core events 0~1 → LOW
core events 2~3 → MID
core events 4~5 → HIGH
core events 5   → HIGH + all_servants_complete
```
