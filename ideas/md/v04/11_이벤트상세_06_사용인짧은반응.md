# GGB v0.4 이벤트 상세 06: 사용인 짧은 반응

## 1. 목적과 공통 규칙

짧은 반응은 호감도 작업을 별도 과업처럼 보이게 하지 않고, 반복되는 일상에서 사용인이 이전 루프를 기억한다는 사실을 드러낸다.

- 1회 20초~90초.
- 메인 퍼즐의 정답·진행 플래그를 지급하지 않는다.
- 같은 이벤트는 최초 장면 이후 1문장 축약 반응으로 교체한다.
- `bond`는 신뢰와 애착, `alert`는 주인공의 탈출 행동을 경계하는 정도다.
- 사용인은 이전 기억을 보유하지만 관리 프로토콜과 상호 감시 때문에 정답을 직접 말하거나 물리적으로 항상 저지할 수 없다.
- 조건이 겹치면 `메인 진행 반응 > 미확인 짧은 반응 > 반복 반응` 순서로 재생한다.

## 2. 에드가

### EDGAR_S1 같은 꿈

| 항목 | 내용 |
| --- | --- |
| 위치·시간 | 중앙홀, A2 이후 아침 |
| 선행 조건 | `notebook_persistence_confirmed`, 미확인 |
| 목표 | 에드가가 루프를 기억한다는 첫 정서 단서 제공 |
| 상호작용 | “어젯밤 꿈”을 묻거나 침묵한다 |
| 성공·실패 | 대화를 끝내면 완료. 실패·진행 차단 없음 |
| 힌트 | 에드가가 어제와 같은 찻잔 흠집을 먼저 가린다 |
| 관계 변화 | 질문 `bond +1, alert +1`, 침묵 변화 없음 |
| 색상 반응 | 장갑 아래 남색 수직선이 한 번 닫힘 |
| 감각 연출 | 낮은 시계음, 말보다 반 박자 늦는 입 모양 |
| 구현 메모 | `EDGAR_S1_seen`; 35~55초 |

대사 기준:

> “꿈은 기록하지 않는 편이 좋습니다. 특히 되풀이되는 꿈은요.”

경계 단계별 문구:

| alert | 문구 성격 | 예시 |
| --- | --- | --- |
| 0~1 | 업무 확인 | “이 시간에 독서라면, 짧게 끝내시는 편이 좋겠습니다.” |
| 2~3 | 우회 경고 | “기록은 오래 들여다볼수록 사람을 닮습니다. 아가씨께는 권하지 않습니다.” |
| 4~5 | 의심 표면화 | “같은 책을 매번 다르게 찾으시는군요. 우연이라고 적어 두겠습니다.” |

에드가는 대화를 끝내면 반드시 퇴장한다. 경계 문구는 다음 루프의 태도를 바꾸지만 B2·J1 접근을 잠그지 않는다.

## 7. 우선순위와 중복 방지

```text
필수 컷신
> 퍼즐 결과 반응
> 핵심 관계 이벤트
> 미확인 짧은 반응
> 반복 한 줄 반응
> 공통 오브젝트 반응
```

- 한 공간 진입에서 자동 대화는 최대 1개.
- 짧은 반응을 연속 두 개 본 뒤에는 최소 한 번의 자유 조작 구간을 둔다.
- `bond` 최대치는 핵심 관계 이벤트 전에도 잠기지 않지만, 짧은 반응만으로 `core_event_complete`가 되지는 않는다.
