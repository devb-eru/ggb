# GGB v0.4 이벤트 상세 03: 메인 퍼즐

## 1. 원칙

- 정답 고정, 무작위 없음.
- 난이도는 교차 단서와 표현 변환에서 발생.
- 반사 신경보다 관찰·가설·검증 요구.
- 색상은 출처 확인 보조이며 단독 정답이 아님.
- 비가역 실패는 B3-B, C4, D1에만 사용.
- 하드 실패 후 영구 정보 또는 검증 부분이 반드시 증가.

## 3. B3-A 시계망 배선 탁본

### 목표

네 시계의 배선 탁본을 회전·반전해 실제 연결을 복원한다.

자료:

- 침실 시계 뒷면.
- 대응접실 하단.
- 서재 벽 균열.
- 서쪽 대시계 점검함.

정답:

```text
침실 시계 = 망에서 단절
대응접실 = 기준
서재 = 중계
서쪽 대시계 = 출력
```

서재 탁본만 뒤집어야 구리선이 연결된다.

색 보조:

- 에드가 남색 잠금선은 점검함 소유권을 표시.
- 배선 정답 자체는 구리선 홈과 진동 방향으로 판정.

힌트:

1. 끊긴 가장자리 확인.
2. 진동이 들어오고 나가는 방향 비교.
3. 서재 탁본의 종이 뒷면 강조.

## 4. B3-B 역할·위상 설정

### 입력

| 슬롯 | 정답 |
| --- | --- |
| 기준 | 대응접실 |
| 중계 | 서재 |
| 출력 | 서쪽 대시계 |
| 제외 | 침실 |
| 위상 | `+1` |

단서:

- 정상 종은 열두 번.
- J1 마침표 뒤 떨어진 잉크점.
- XII 뒤 이름 없는 홈.
- 본 종이 끝난 뒤 서쪽 점검.

`약한 시험 진동`은 연결만 검증하고 위상은 검증하지 않는다.

하드 실패:

- 실제 저녁 작동 후 오답이면 봉인핀이 꺾임.
- 시계망 당일 잠금.
- 맞은 역할 슬롯은 영구 검증.
- 다음 루프 BSHORT로 B3-B부터 재시도.

성공:

- 열두 번 뒤 열세 번째 금속음.
- B4 파형 획득.
- 색 서명 다섯 개가 한 프레임 시계망에 겹치지만 정체는 미공개.

## 5. C3 중성 세정제 제약식

### 원자료

```text
W + S + A = 8
A = 2S
W = S + A + 2
```

유일해:

```text
증류수 W = 5
안정제 S = 1
세정 원액 A = 2
순서 = 물 → 안정제 → 원액
```

오답:

| 유형 | 반응 |
| --- | --- |
| 총량 | 넘침·부족 |
| 비율 | 시험지 변색 |
| 순서 | 결정 발생 |

모든 오답은 조합대에서 즉시 재시도한다. 색 변화는 시험지 문양과 `산성/염기성` 텍스트를 병행한다.

## 6. C4 파형·거울 중첩

### 파형 대응

| B4 구간 | 거울 |
| --- | --- |
| 길고 낮은 진입파 | 직선 |
| 짧은 두 반사파 | 분기 양쪽 |
| 닫히는 잔류파 | 고리 |

중첩 정답:

```text
좌우 반전 없음
90° 시계 방향
열세 번째 파동을 거울 하단 진동점에 정렬
```

닦기 경로:

```text
직선
→ 분기 짧은 쪽
→ 분기 긴 쪽
→ 고리 시계 방향
```

마른 천 시험은 가역적이다. 세정제를 묻힌 뒤 실제 경로 오류는 코팅을 경화시킨다.

에드가 개입:

- 일반 천으로 점검 유도.
- 일지 질문으로 대화 소모.
- 정확한 순찰 공백 대기.

관계가 높으면 개입이 줄지만 정답은 동일하다.

## 7. C5 퍼즐 출력

획득:

- 직선·분기·고리의 반사 원도.
- 서쪽 대시계, 온실 유리, 침실 창문 기준점.
- 지하 좌표.
- 다섯 인격 색 채널.

색 채널은 진단 패널의 소유자 기록이다.

```text
남색 수직선
주황 대각선
검정·연두 맥박
흰색·연노랑 꽃잎
보라 이중 액자
```

MARA2_S2는 보라 채널의 체크섬이 다른 네 채널보다 짧다는 사실만 알려준다.

## 8. D0-A 반전 저택 도면

자료:

- C5 회로 투명지.
- 저택 평면도.
- J3 지하 좌표.

정답:

```text
C5 투명지 좌우 반전
→ 90° 반시계 회전
→ 대시계 기준점 고정
→ 온실·침실 기준점 확인
```

출력:

| 축 | 깊이 |
| --- | --- |
| 직선 | 2 |
| 분기 | 1 |
| 고리 | 3 |

색상 필터는 C5 채널을 숨겨 회로 원도만 보게 하는 보조 기능이다. 패턴 필터로 동일 기능 제공.

## 9. D1 세 축의 순서·깊이

정답:

```text
직선축 깊이 2
→ 분기축 깊이 1
→ 환형축 깊이 3
→ 중앙 손잡이 시계 방향 반 바퀴
```

축을 밀기 전 깊이는 가역적이다. 민 뒤에는 압력핀이 내려와 되돌릴 수 없다.

실패 기록:

- 순서 오류.
- 깊이 오류.
- 잠금 단계.
- 검증된 이전 축.

다음 루프에는 D0-A 결과와 검증 축을 유지한다.

## 10. D4 연동 태엽 심장

목표 상태:

```text
초기 [0,0,0]
목표 [외곽 1, 중간 2, 안쪽 3]
```

손잡이:

| 조작 | 효과 |
| --- | --- |
| A | 외곽 +1, 중간 +1 |
| B | 중간 +1, 안쪽 +1 |
| C | 안쪽 +1 |

최소해:

```text
A 1회 → B 1회 → C 2회
```

목표 문양:

- 외곽 XIII.
- 중간 C5 분기 접점.
- 안쪽 J3 최하단 심장.

기동:

1. XII 정상 기동.
2. `STABLE` 표시.
3. 완료 뒤 숨은 +1 보조 레버.
4. XIII 완성.
5. `CAMOUFLAGE FILTER OFF`.
6. D5.

다른 긴 조작으로 같은 목표 상태를 만들면 허용한다.

## 11. E3_5 마라 2 원본 대조

### 입력

서로 다른 시점의 단체 초상화 3장. 다섯 색·문양 채널이 겹쳐 있다.

### 단계

1. 색 또는 문양 필터로 다섯 출처 분리.
2. 보라 이중 액자·3음 신호 조각 3개 추출.
3. 조각을 시간순으로 중첩.
4. 체크섬 12칸 중 비어 있는 3칸 확인.
5. 다른 사용인 기록에 같은 3칸이 보존됐음을 발견.
6. 마라 2가 자기 저장 영역을 양도한 사실 확인.

### 선택

| 선택 | 결과 |
| --- | --- |
| 감정 주석을 원본에 병합 | bond +2, alert +1, 감정 기억 선명 |
| 원본과 주석을 별도 보존 | bond +1, alert -1, 기술적 안정 |

둘 다:

- `REC_MARA2` 획득.
- 메인 진행 차단 없음.
- F2·엔딩 대사 변형.

오답은 로컬 재조작. 색을 끄면 이중 윤곽과 3음 파형으로 해결.

## 12. F0-A 방 피드백 회로

원형 슬롯:

| 방향 | 방 |
| --- | --- |
| 북 | 온실 |
| 동 | 주방 |
| 남 | 침실 |
| 서 | 서재 |

주 흐름:

```text
온실 → 주방 → 침실 → 서재 → 코어
```

피드백:

```text
서재 → 온실
```

방 타일의 물질 실선과 데이터 점선을 모두 연결해야 한다.

| 방 | 정답 | 오답 |
| --- | --- | --- |
| 온실 | 외부 대기 | 꽃향기 연출 |
| 주방 | 생명 유지 유체 | 차 메뉴 |

정답은 현실의 주인공을 유지하는 데이터다. 오답은 고딕 표현 데이터다.

## 14. F0-C 세 자료 중첩

자료:

- B4 파형.
- C5 회로.
- D4 다섯 포트 잔상.

정답:

| 자료 | 조작 |
| --- | --- |
| D4 | 고정 |
| C5 | 좌우 반전 + 90° 반시계 |
| B4 | 180° 회전 |

완전 일치:

- 열세 번째 파동과 빈 포트 중첩.
- 분기선이 네 채널과 빈 포트 연결.
- 고리가 빈 포트를 닫음.

## 15. F0-D 기록 역할 분류

| 슬롯 | 기록 |
| --- | --- |
| CREATOR | 아버지 일지 |
| CUSTODIAN | 에드가 암구 |
| RESIDENT | 다섯 연구원 기록 묶음 |
| SYSTEM | D4 복구 명령 |
| SUBJECT | 주인공 수첩 |

색상 서명은 RESIDENT 기록 안의 출처를 확인할 뿐 역할 슬롯은 기록 내용으로 추론한다.

마라 2 미완료:

- `REC_MARA2` 대신 보라 이중 윤곽의 익명 인덱스.
- 역할 분류와 SUBJECT 해법에 영향 없음.

## 16. F0-E 과거·현재 인증

1. A1 수첩 표시 재현.
2. 과거 자기 연속성 확인.
3. 현재 권한 문장 선택.

정답:

```text
아직 정하지 않았다. 내가 직접 정할 것이다.
```

완료:

```text
SUBJECT AUTHORITY RESTORED
FINAL DECISION: UNSET
```

SUBJECT 표현은 연필선, 종이 질감, 주인공 필기음이다. 사용인 색을 사용하지 않는다.
