## 문제

- 검은 거울이 대응접실 내부 또는 거울 복도로도 표기된다.
- 공식 위치는 대응접실 남쪽 별도 `M1_MIRROR_GALLERY`다.
- E3_5는 지도에 없는 `NORTH_ARCHIVE_COLOR_ROOM`을 사용한다.

## 권장 해결

```text
C0~C5: M1_MIRROR_GALLERY
E3_5 전반: H0_COLOR_SEPARATION
E3_5 후반: H0_PERSONALITY_ARCHIVE
```

## 해결 체크리스트

- [x] `거울 복도`를 `거울 회랑`으로 통일.
- [x] 대응접실 내부 거울로 오해되는 문장 교정.
- [x] E3_5의 실제 위치 ID 2개 등록.
- [x] 모든 `location_id`가 05 레지스트리에 존재하는지 검사.
