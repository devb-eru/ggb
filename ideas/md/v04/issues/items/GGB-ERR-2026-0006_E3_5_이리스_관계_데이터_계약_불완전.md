## 문제

- E3_5의 `interaction_nodes`가 빈 배열이다.
- `iris_confession_state`가 `17`의 영구·파생 상태에 없다.
- E3_5 두 결과의 branch variant가 정의되지 않았다.

## 권장 해결

- E3_5 퍼즐·대화·보존 선택 노드를 등록한다.
- `iris_confession_state`를 계산 프로퍼티 또는 저장 enum으로 명시한다.
- `merged`, `separated` branch result를 데이터 계약에 추가한다.

## 해결 체크리스트

- [x] E3_5 시작부터 E_HUB 복귀까지 노드 연결.
- [x] 이리스 고백 6상태 enum 등록.
- [x] F2와 엔딩이 동일한 고백 판정을 읽는지 확인.
